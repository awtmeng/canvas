# 第三节课绘制三次方贝塞尔曲线
## 绘制二次方贝塞尔曲线方法

```
quadraticCurveTo(double cpx, double cpy, double x, double y)
cpx, cpy 表示曲线控制点
x，y表示描点
```

```
bezierCurveTo(double cpx, double cpy, double cp2x, double cp2y, double x, double y)
cpx, cpy, cp2x, cp2y 表示曲线控制点
x，y表示描点
```
